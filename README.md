# conf-linux

My personal linux config. Contain some customized settings. Including:

* Vim
* Terminator
* Git
* Mutt

Prerequisites
-------------

* vim
* terminator
* git
* mutt
* fetchmail
* procmail

Recommended to install
----------------------

* vim-gtk

Usage
-----

	git clone git@github.com:dstw/conf-linux.git
	cd conf-linux

### Vim

	cp .vi* ~/

### Terminator

	cp .config ~/

### Git
	
	cp .gitconfig ~/

### Mutt

	cp .fetchmailrc .procmailrc .muttrc .forward ~/
