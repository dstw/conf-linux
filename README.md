# conf-linux

My personal linux config. Contain some customized settings. Including:

* Vim
* Terminator

Prerequisites
-------------

* vim
* terminator

Recommended to install
----------------------

* vim-gtk

Usage
-----

	git clone git@github.com:dstw/conf-linux.git
	cd conf-linux

### Vim

	cp .vi* ~/

### Terminator

	cp .config ~/
